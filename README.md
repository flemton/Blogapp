# 📖 [Blogapp] <a name="about-project"></a>

**[Blogapp]** is a web application built with Ruby on Rails that allows users to create, read, update, and delete blog posts. Users can sign up, log in, and share their thoughts with the community by creating blog posts. The app uses a PostgreSQL database to store user information and blog posts.

## 🛠 Built With <a name="built-with"></a>
- Ruby on Rails
- PostgreSQL

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd Blogapp
  git clone git@github.com:flemton/Blogapp.git
```

### Install

Install this project with:


```sh
  cd Blogapp
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rails test
```
